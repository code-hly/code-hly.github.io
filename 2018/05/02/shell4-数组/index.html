shell数组

发表于 2018-05-02 | 分类于 shell | | 热度: ℃
字数统计: 1.3k | 阅读时长 ≈ 5

Shell并且没有限制数组的大小，理论上可以存放无限量的数据。Shell数组元素的下标也是从 0 开始计数。获取数组中的元素要使用下标[ ]，下标可以是一个整数，也可以是一个结果为整数的表达式；当然，下标必须大于等于 0。 常用的Bash Shell只支持一维数组，不支持多维数组。
